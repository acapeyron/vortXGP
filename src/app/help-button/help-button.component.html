Purpose of this project

Which mode do I choose ?

The VortX project is ... (vortex core detection, for that we focused on simulating a wind turbine's blade rotating and processed the result through machine learning algorithms to detect the vortices cores.)
There are two modes available:
-The easy mode is for every user possible. No need to have experience in the vortex topic or computer engineering. The point is to allow everyone to discover how we worked and what we achieved and play around with the results.
-The advanced mode is reserved for users with a background on the topic. It allows for more room on user input and customization of the user's experience.

Welcome to the easy mode

For this mode we have run 12 different simulations of a wind turbine's blade rotating with different parameters. This page allows you to have a look at all these simulations and compare them. Once you are ready, you can go to the next page where the results will be displayed.

Press the Tutorial button if you want to follow the tutorial or Skip this part.
